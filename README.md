# Mapa de Capacidades - Plataforma Web

Aplicación web Front-End desarrollada para visualizar y gestionar el mapa de capacidades del proceso operativo de elaboración de imanes personalizados. Este proyecto traduce la arquitectura de negocio en una interfaz interactiva y estructurada.

**Autor:** Homero Mackenzie - Chief Marketing Officer  
**Organización:** Memorias & Recuerdos  

## Tecnologías Utilizadas
* **Framework:** React con Next.js (App Router)
* **Lenguaje:** JavaScript
* **Estilos:** Tailwind CSS
* **Testing:** Jest + React Testing Library

## Requisitos Previos
* Node.js (versión 18 o superior)
* Gestor de paquetes: `npm` o `yarn`
